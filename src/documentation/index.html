Element variable

isData

isData indique si l'élément variable est structurel, c'est à dire qu'il contient des données qui sont liées à la struture de la page, ou bien des données

L'édition des données permet de générer des pages sur le modèle du teemplate, tandis que l'édition de données structurelles correspond à une modification du template

ElementId

L'id d'un élément variable doit permettre d'accéder à l'élémétnt du DOM

Signature

La signature d'un template doit permettre d'éviter le stockage de deux templates équivalents.
